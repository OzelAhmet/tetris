# Tetris Game

A simple Tetris game that runs on terminal.

Requires `Python 3.6` or higher.

Just run `python3 tetris.py`

## Keys

* **Left:** Shift block left
* **Left:** Shift block right
* **Down:** Shift block bottom
* **Up:** Change the block orientation
